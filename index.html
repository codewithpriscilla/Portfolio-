Projects

Pawbnb

PawBnB is like Airbnb but for dogs. It was created using Django framework for the back end and React for the front end. Similar to Airbnb you can browse homes without an account, however you can also create an account or log in to an existing one.

JobTrackr

JobTrackr is a website designed to keep track of the jobs you've applied to. It was designed and developed by myself and 2 other developers using Mongoose, Express, React, and Node.js

Forever Friends

'Forever Friends', my first project using an API and React, is a dog and cat adoption website. Each adoption page gathers images from the APIs then once a pet is clicked, their profile is brought up with details about the pet.
